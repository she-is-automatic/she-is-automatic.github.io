【装机】IDM-Activation-Script 下载激活 Internet Download Manager (IDM)

Posted on 2024-08-22 In Tools
开源的IDM-Activation-Script，下载并激活idm，支持在线更新。

WHY IDM？

Windows下载神器。最常用的功能：

  • 多线程下载加速
  • 视频资源嗅探

工具

IDM-Activation-Script：支持IDM官方下载与激活，支持在线更新。

步骤

官方推荐powershell教程：

  • Right-click on the Windows start menu and select PowerShell or Terminal (Not CMD).
  • Copy-paste the below code and press enter *iex(irm is.gd/idm_reset)
  • You will see the activation options, follow the on-screen instructions.
  • That’s all.

step1：在powershell中输入iex(irm is.gd/idm_reset)，允许权限，出现如下界面：

step2：输入4，浏览器自动跳转到IDM官方下载地址，下载IDM安装包，安装后显示未注册版本（已经安装idm且仍在30天试用期内可跳过此步），浏览器中会自动安装IDM拓展，选择启用拓展。

step3：回到脚本菜单，选择[1]Activate，在如下界面选择[9]Activate。在激活过程中会下载几张图片，激活完成后显示绿色的The IDM Activation Process has been completed.，即为激活成功。可在IDM中查看注册信息。
